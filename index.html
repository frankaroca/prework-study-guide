HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The footpage contains information about the autor of the website, year created or other information
  • The attribute id in HTML is a way to identify elements
  • In Head of the HTML code will be some information not visible for the user a exception of the title

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can select color for fonts or other objects using HTML Hexadecimal codes or using semantic references
  • You can add more than one property to any objects, just separate those using comas
  • The period preceding a class is a class selector and any HTML element in that class will gain those properties

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switches you out to the branch main
  • git code .: opens the code editor for default
  • git cd folder name: takes you into the folder or just cd it takes you to the main file folder
